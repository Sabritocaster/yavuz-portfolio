# Yavuz Dağdelen — Design Portfolio

### [yavuzdagdelen.com](https://www.google.com/search?q=https://yavuzdagdelen.com)

This is the official portfolio website for **Yavuz Dağdelen**. The site serves as a gallery for his works and a blog.

## Homepage Animation

This project utilizes [Amanda Ghassaei's](https://github.com/amandaghassaei) [GPU-IO](https://github.com/amandaghassaei/gpu-io) library to render fluid, real-time simulations directly in the browser.

---

## Technical Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Animation Engine:** [GPU-IO](https://github.com/amandaghassaei/gpu-io)
* **Typography:** Area Normal by Blaze Type
